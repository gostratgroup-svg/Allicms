# AlliCMS Assets

Place brand files here so Vite can serve them from `/assets/...`.

Suggested logo names:

- `allicms-logo-main.png` or `allicms-logo-main.svg`
- `allicms-logo-alt.png` or `allicms-logo-alt.svg`

The main app logo is used in the sidebar. The alternate indigo logo is for other brand use cases.
