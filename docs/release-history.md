# AlliDesk Release History

## Version 0.1

Date: 2026-07-06

Summary: Foundation release for moving AlliDesk from prototype toward production.

Included:

- Brand renamed to AlliDesk.
- Supabase connected.
- Architecture completed.
- Phase 1 database migration completed.

Notes:

- The app still uses prototype/localStorage workflows for user-facing product areas.
- Live Supabase workflow integration has not started yet.
- No patient, booking, clinical, finance, document, payment, or storage production tables are connected to the app yet.
